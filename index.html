skills.txt
this site was built using modern HTML5, CSS3, and JS.

windows are organized in a hierarchy of <div> tags in HTML, and the animations are purely handled by CSS transitions and pseudo-classes. semantic HTML tags are used as appropriate. window dragging, hover effects, and theme toggling is implemented in JS using event listeners. this site is responsive and works well on mobile devices, the experience is slightly different but still functional. implementing this site is a marker of my progress in this course and a demonstration of my skills.
